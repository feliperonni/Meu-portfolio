# Meu Portfólio

Olá! 👋  
Esta é a primeira versão do meu portfólio pessoal, feito apenas com **HTML**. Aqui você pode conhecer um pouco sobre mim, meus hobbies e entrar em contato.  

## 📌 Acesse meu portfólio online
[Veja meu portfólio](https://feliperonni.github.io/Meu-portfolio/)

## 🛠️ Tecnologias usadas
- HTML semântico

## 📂 Estrutura do projeto
- `index.html` → página principal do portfólio
- `img/` → pasta contendo imagens usadas no site
- `<header>` → contém título, descrição, imagem e menu de navegação
- `<nav>` → menu com links internos para as seções do site
- `<main>` → conteúdo principal, dividido em seções:
  - `<section id="sobre">` → sobre mim
  - `<section id="hobbies">` → meus hobbies
  - `<section id="contato">` → informações de contato
- `<footer>` → rodapé com direitos autorais
